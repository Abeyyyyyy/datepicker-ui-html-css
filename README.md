# Datepicker UI (HTML & CSS)

This project is a simple **static datepicker UI** built using only HTML and CSS.

## 🚀 Features
- Date input with placeholder
- Calendar icon button
- Calendar popup using absolute positioning
- Grid layout for days and dates
- No JavaScript involved

## 🛠️ Tech Stack
- HTML
- CSS

## ▶️ How to Run
1. Clone the repository
2. Open `index.html` in your browser
